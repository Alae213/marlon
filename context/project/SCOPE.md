# Project Scope

> Generated by setup wizard. Updated from PRD.md analysis.

## In Scope — v1 (MVP)

### Core Platform
- Multi-tenant store creation (one user → multiple stores)
- Store dashboard with subscription status badges
- Store slug uniqueness validation

### Store Editor
- Product management: add, edit, delete, archive, reorder
- Rich text product descriptions
- Product variants (size, color, custom)
- Multiple product images with drag-reorder and cropping
- Site content editor: navbar, hero section, footer
- Delivery pricing per wilaya (stopdesk + domicile)

### Order Management
- Order list with filters (status, date range)
- Notion-style table view
- Order status machine: new → confirmed → packaged → shipped → succeeded
- Status actions: confirm, package, ship, cancel, block, hold
- Call logging with outcomes (answered, no answer, wrong number, refused)
- Order detail panel with inline editing
- Internal notes (merchant-only)
- Audit trail / changelog

### Public Storefront
- Store catalog page with product grid
- Product detail page with variant selection
- Shopping cart with localStorage persistence
- Checkout flow with Algeria-specific fields
- Real-time delivery cost calculation
- Order confirmation page

### Billing System
- 50-order free tier (rolling 30-day window)
- 2000 DZD/month paid tier (manual payment)
- Locked state with masked customer data
- Payment webhook processing

### Integrations
- Clerk authentication (Google OAuth)
- Convex backend with real-time subscriptions
- Delivery API integration (ZR Express, Yalidine)
- Payment provider abstraction (configurable)

## Out of Scope — v1 (Deferred)

- Store templates / themes / color pickers / CSS editors
- Custom domains
- Arabic / French UI language support
- Super-admin dashboard
- Store suspension / merchant approval
- Analytics / charts
- Mobile app
- Real-time delivery tracking maps
- Online payments (COD only)
- Inventory / stock alerts
- Customer accounts / order history
- Marketing tools (discounts, email campaigns, referrals)
- Public merchant API
- White-label / agency plans
- Auto-renewal billing

## Never In Scope

- International shipping
- Multi-currency support beyond DZD
- Payment gateways outside Algeria
- Inventory management system
- Point-of-sale integration
